About this app

How to Use with 5 buttons

  • Navigate the calculator by using the w,a,s,d, and k keys
  • w moves up, a moves left, s moves down, d moves right
  • k will select the current item
  • Test results

How to Use with 3 buttons

  • Navigate the calculator by using the a, d, and k keys
  • a moves left and will jump to previous row
  • d moves right and will jump to next row
  • k will select the current item

Test Results

Ran a time trials on performing the fallowing task while 5 buttons vs 3 buttons: 10+10 = 1+2+1+4+1 = 7+5+3 = 167+754 = 12*9+8 = Watching the test subject perform these tasks and knowing they are timed resulted in a few mistakes. The subject will select the wrong key by mistake a few more while using the 3 buttons then the 5. Over all the 5 buttons had less errors and performed quicker. Each of these test was given to each of my Roomates (2 roomates) and the results were similar amoung each. For future there should be some mechinism to revers the selection of a mistake rather then starting over.
